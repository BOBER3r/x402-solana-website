# x402 Payment Channels Website

Official website for x402 payment channel protocol on Solana. Built for the Solana hackathon submission.

## Overview

This website showcases the x402 protocol - a payment channel implementation for HTTP APIs on Solana that enables micropayments with 99.8% cost savings compared to traditional payment processors.
